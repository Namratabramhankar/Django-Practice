STEPS FOR DJANGO
1. From the cmd reach the folder where the project is to be created.
2. django-admin startproject projectname
3. python manage.py runserver

TO CREATE APP
1. python manage.py startapp articles.
	This creates a folder with name "articles" in the main directory of the project folder.

MIGRATIONS
1. python manage.py makemigrations
2. python manage.py migrate

TO CREATE ADMIN DATA:
1. python manage.py createsuperuser
This will generate the user data for login of admin page.
